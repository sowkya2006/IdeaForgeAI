# 🚀 IdeaForge AI

IdeaForge AI is an AI-powered project blueprint generator that transforms project ideas into complete development plans.

## Features

- 📌 Idea Analysis
- ⚡ Benefits & Challenges
- 💡 Innovation Analysis
- 🛠 Feature Suggestions
- 🚀 MVP Planning
- 🏗 Tech Stack Recommendation
- 🏛 System Architecture
- 🗄 Database Design
- 📅 Development Roadmap
- 💰 Cost Estimation
- 📊 Market Analysis
- 🏆 Competitor Analysis
- 💸 Revenue Model

## Tech Stack

- Python
- Streamlit
- Groq API
- Llama 3.3 70B

## Installation

```bash
git clone <repository-url>
cd IdeaForgeAI
pip install -r requirements.txt
```

## Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```

## Run Application

```bash
streamlit run app.py
```

## Deployment

The application can be deployed on:

- Hugging Face Spaces
- Streamlit Community Cloud
- Render
